Mock Test Generator: Auto Test Creation & Evaluation System

Description: This web-based tool allows students and educators to generate and take mock tests across various subjects. The platform supports random question generation, time-bound sessions, automated scoring, and result analysis to simulate real exam conditions.

Key Features:

  • Dynamic test generation from question bank
  • MCQ, true/false, and short answer formats
  • Timer and auto-submit functionality
  • Instant scoring and answer review
  • Admin panel to manage questions and subjects

Technologies Used: HTML, CSS, JavaScript, PHP, MySQL
