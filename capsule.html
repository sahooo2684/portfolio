Capsule: Productivity Tracker App

Description: Capsule is a personal productivity tracking app designed to help users monitor their daily tasks, habits, and time logs. It features visual analytics for performance and habit streaks.

Tools Used: Android (Java), SQLite, XML, SharedPreferences

Features:

  • Daily task creation and completion tracking
  • Habit tracking with streak notifications
  • Time logging with pie chart summaries
  • Dark/light mode support

Status: Beta version complete, in testing phase
